## Chapter 26: Subdivisions

**41 sections across 2 articles - VERY IMPORTANT for land division questions**

### Key Sections by Topic

#### Authority & Purpose (26-11 through 26-13)
- Legal authority for subdivision regulations
- Purpose and scope

#### Application & Fees (26-14, 26-15)
- Application requirements
- Fee schedules

#### Exemptions (26-15.1 through 26-15.3) - FREQUENTLY ASKED
Common exemptions from full subdivision process:
- **Lot line adjustments** (boundary changes between existing parcels)
- **Certificate of Compliance** (legalizing existing parcels)
- **Certain divisions** (government, utilities, etc.)

#### Definitions (26-21 through 26-21.32)
Important defined terms:
- Lot, parcel, subdivision
- Tentative map, parcel map, final map
- Design standards terminology

**When to use**:
- "Can I split my property?"
- "What's a lot line adjustment?"
- "Do I need a subdivision map?"
- "Is my parcel legal?"

## How to Use County Code Tools

### Finding Relevant Sections

**Tool**: `search_county_code`
```
query: "lot line adjustment"
chapter: "26" (optional - limits to specific chapter)
max_results: 10
```

### Getting Full Section Text

**Tool**: `get_county_code_sections`
```
section_ids: ["28.21.20", "28.21.30"]
```

Retrieves complete text including tables. For use tables (what permits are needed), this returns formatted markdown tables.

### Listing Available Sections

**Tool**: `list_county_code_sections`
```
chapter: "26"
```

Returns all section IDs and titles in that chapter - useful for browsing.

## Important Notes

1. **County Code only applies to unincorporated areas** - Cities have their own codes

2. **Chapter 28 is huge** - 166 sections. Start with search, then retrieve specific sections

3. **Tables in 28.21.20** - The use/permit tables are crucial for answering "what permit do I need?" questions

4. **Cross-reference with General Plan** - Code implements General Plan policies. For policy context, also search the General Plan.

5. **Not all county code is available** - Only chapters 19, 23, 24, 26, 26.5, 28, 30, 31 are in the database. For other chapters (e.g., building code, health regulations), direct users to the full code online.
